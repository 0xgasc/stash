## ðŸ§  The Genius of Perceived Value Pricing

### **From PermaStore Doc - The "Zora Model"**

**Key Insight:**
> "Storing a 3GB video costs us the same per GB as storing a 3MB image, but we charge 8x more for 'video tier' access."

**How we apply this:**

1. **File Size Limits = Psychological Anchoring**
   - Starter: 100MB/file = "for documents and photos"
   - Pro: 500MB/file = "professional tier"
   - Creator: 3GB/file = "premium for video creators"
   - **Reality:** All cost us $1/GB regardless!

2. **"Priority" Features (Zero Cost)**
   - Pro gets "priority uploads" = same infrastructure
   - Creator gets "NFT API" = just CORS-enabled endpoint
   - Custom domain = costs us $0, perceived value $10/mo
   - Analytics = simple query, huge perceived value

3. **Bundling Strategy**
   - Each tier gets previous tier features PLUS:
   - Pro = Starter + custom domain + priority
   - Creator = Pro + video + API + analytics
   - **Upsell ladder**

### **Example Revenue Math:**

**Video Creator uploads 100GB:**
- **Charged:** $30/mo + $0 overage (200GB allowance)
- **Our Cost:** $100 one-time (100GB @ $1/GB)
- **Month 1:** -$70 (loss)
- **Month 2:** +$30 (profit)
- **Month 3:** +$30 (profit)
- **Month 4+:** Pure profit ($30/mo forever)
- **After 4 months:** $10 profit, then $30/mo = **300% CAGR**
